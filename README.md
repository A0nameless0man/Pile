## 我想提一下的有趣的东西

### 自动提交脚本

因为这个仓库里面东西多又杂，不是很想每次都写commit。
这个脚本接受一个参数作为commit的内容，没有的话会填当前时间。commit然后自动push。

### 校验和计算

自动计算一个目录下所有文件的校验和，多线程的。

输出顺序是按照路径的字典序排序的。
